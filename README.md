# System Analysis Portfolio

Учебные кейсы по системному анализу.

## Кейс: система заказа пиццы
Полное описание одной предметной области разными артефактами —
чтобы было видно, как держу контекст целиком.

- **Use Case** — роли и варианты использования
- **Class / ER** — модель данных и связи сущностей
- **BPMN** — процесс оформления заказа с ветками и обработкой ошибок
- **SQL** — запросы к тем же сущностям (топ клиентов, отток, средний чек)
- **ТЗ** — функциональные и нефункциональные требования

### Стек
UML · BPMN · SQL · REST · Python · EN B1
